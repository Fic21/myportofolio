Monitoring Suhu Ruangan Dengan Aplikasi

Monitoring Suhu Ruangan dengan Aplikasi ini adalah tugas yang diberikan dosen pada saat saya berkuliah ketika itu kami para mahasiswa diberikan tugas untuk memonitoring suhu dengan cara membuat dengan menggunakan Visual Basic 6.0 tetapi pada saat itu saya pada laptop saya hanya bisa install Visual Studio 2013 tidak dengan instalan Visual Basic 6.0. saya meminta saran ke dosen terkait dengan penggunaan aplikasi yang tidak sesuai dengan ketentuan tugas yang dia berikan, lalu dengan negoisasi tersebut akhirnya saya diperbolehkan menggunakan Visual Studio 2013.

Hal yang paling berkesan pada saat itu. saya belum pernah melakukan percobaan komunikasi data dari hardware ke software pada saat itu dosen pun tidak pernah mengajarkan sebelumnya. Pada akhirya saya mencari cara agar bisa menampilkan suhu ke software, setelah melalui riset yang telah saya lakukan saya memilih Mikrokontroler arduino Uno dan sensor suhu LM35

Pada saat perancangan saya terlebih dahulu membuat program di mikrokontroler agar dapat menerima data dari sensor suhu yang akan dikirim ke komputer yang akan menampikan pada aplikasi yang telah dibuat. setelah pembuatan program pada mikrokontroler saya melanjutkan pemograman pada Visual Basic agar dapat menerima dan menampilkan data dari perangkat hardware

Dari pengalaman tersebut saya bisa memahami bagaimana sistem komunikasi data serial antara komputer dengan mikrokontroler dengan pengalaman tersebut saya bisa memahami cara kerja pengiriman data serial dari mikrokontroler ke komputer
